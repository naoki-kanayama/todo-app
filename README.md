# ToDoアプリ

## 概要
オリジナルアプリ制作の中でJavascriptの理解が浅いと思ったため、「JavaScript基礎入門」を参考にToDoアプリの制作を行いました。

# 開発状況
## 開発環境
  * 開発言語
    * JavaScript
  * 開発ツール
    * Visual Studio Code

# DEMO
入力フォームを送信することでTodoリストが更新されます。
また、リスト更新後に優先順位の変更、登録日・優先順位での並べ替えも行うことができます。
